# Photobooth app
A simple photobooth script that has the following features
1. Selection for how many photos you want
2. Selection for custom frames
3. Interact with frames
4. Print
5. Extendable with features like email
## How to run this app
```
pyenv activate venv
```

v0.1 flow
1) User come in and press start
2) 30 seconds countdown
3) then take 4 photos
4) then ask user to choose the photos they want (min 2, max 3)
5) give a preview of the strip of 3 photo or strip of 2, choose number of prints (even)
